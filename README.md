# Task Manager App

Full Stack Task Manager built with:

- NestJS (Backend)
- Next.js (Frontend)
- MongoDB
- JWT Authentication

## How to Run

### Backend
cd backend
npm install
npm run start:dev

### Frontend
cd frontend
npm install
npm run dev

Make sure MongoDB is running locally.

## Features
- User Registration & Login
- JWT Authentication
- Protected Routes
- Full CRUD for Tasks
- Filter & Sorting
